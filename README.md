
# Governed Agri-AI Stress Risk Advisor

Streamlit-based governed AI prototype with:
- Policy mode (Research vs Public Service)
- Consent + child protection logic
- Algorithm registry + model card
- Dataset licensing verification
- Fairness diagnostics and heatmaps
- Energy & carbon footprint logging
- Automated Algorithmic Impact Assessment (AIA) JSON report

## How to run

```bash
pip install -r requirements.txt
streamlit run app.py
```
